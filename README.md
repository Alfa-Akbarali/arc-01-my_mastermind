## Usage
How does it work?
This game where the player has to guess a secret code. The secret code is a sequence of 4 unique digits. The player has 10 attempts to guess the code correctly. After each guess, the game provides feedback about how many digits are correct and in the correct position 'well-placed' and how many digits are correct but in the wrong position 'misplaced'.
